## springboot project yunfeigj-devops
### 项目简介
YunfeigjDevops运维管理平台系统基于Spring boot、Spring Cloud、Vue.js、Element UI等前后端技术，采用前后端分离架构实现的管理系统。

YunfeigjDevops实现的主要功能包括:
- 系统登录
- 用户管理
- 工单管理
- 角色管理
- 菜单管理
- 配置管理
- 发布管理
- 监控管理
- 数据监控
- 聚合文档
